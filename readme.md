##  Quick start

1.  **Clone this repository.**

    ```sh
       https://github.com/ferhatkefsiz/payment-form.git
    ```

2. **Install the required dependencies with `npm`:**

    ```sh
       npm install
     ```

3.  **Start Server**

    ```sh
       npm run dev.
    ```
